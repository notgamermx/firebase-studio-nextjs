# **App Name**: Nova

## Core Features:

- AI-Powered Chat: Engage in natural language conversations powered by a local AI, without the need for API keys or external dependencies.
- Example Prompt Suggestions: Get started quickly with pre-defined example prompts to guide users and showcase the chatbot's capabilities.
- User Input and Display: Allows the user to input their message, and see it rendered in the chat window.
- Message History: Save and recall previous messages, but only if requested explicitly by the user. Stored in browser storage.

## Style Guidelines:

- Primary color: Light purple (#A076F9), mirroring the visual identity, to convey a sense of innovation and intelligence.
- Background color: Very light gray (#FAFAFA), to provide a clean, distraction-free conversational experience.
- Accent color: Slightly darker shade of purple (#794BC4), for interactive elements like the citation toggle and the 'New Thread' button.
- Body and headline font: 'Inter' for a modern, clean, and highly readable interface. Also, this matches the HTML from the provided code sample.
- Simple, minimalist icons for actions like attaching files and selecting writing styles.
- A clean and modern layout, mirroring the appearance of sophisticated chat interfaces with a clear separation between the header, chat window, input area, and footer.
- Subtle animations and transitions for user interactions and loading states to enhance user experience.
